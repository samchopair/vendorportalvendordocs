# Vendor Portal Documentation

Welcome to the Vendor Portal documentation! This comprehensive guide will help you understand and effectively submit product information through the Vendor Portal system.

## What is the Vendor Portal?

The Vendor Portal is a web application that enables vendors to submit product information for review and approval. The system facilitates seamless product data collection through Excel templates, batch tracking, and real-time notifications about your submission status.

## System Requirements

### Supported Browsers

The Vendor Portal works best with modern web browsers:

- **Google Chrome** (recommended) - Version 90 or later
- **Mozilla Firefox** - Version 88 or later
- **Microsoft Edge** - Version 90 or later
- **Safari** - Version 14 or later

### Internet Connection

- Stable internet connection required for real-time notifications
- Minimum recommended speed: 5 Mbps download, 1 Mbps upload
- WebSocket support required for live updates

### Screen Resolution

- Minimum resolution: 1280x720 pixels
- Recommended resolution: 1920x1080 pixels or higher
- Mobile responsive design available for tablets

### File Requirements

For Excel uploads:
- **Format**: .xlsx (Excel 2007 or later)
- **Maximum file size**: 10 MB
- **Maximum rows**: 1,000 products per upload
- Excel templates must be downloaded from the Template Builder

## Key Vendor Tasks

### Creating Excel Templates

The Template Builder helps you create properly formatted Excel files:

- Select your product category
- Choose required and optional attributes
- Download a pre-formatted Excel template
- Template includes validation rules and instructions

Learn more: [Template Builder](./vendor/template-builder.md)

### Uploading Products

Submit your product information through Excel uploads:

- Use templates from the Template Builder
- Fill in product details following the format
- Upload files up to 10 MB with 1,000 products
- Receive immediate validation feedback

Learn more: [Product Upload](./vendor/product-upload.md)

### Tracking Submissions

Monitor the status of your product batches:

- View all submitted batches and their status
- Track approval progress in real-time
- Receive notifications when batches are reviewed
- Download feedback and error reports

Learn more: [Batch Tracking](./vendor/batch-tracking.md)

### Managing Products

View and manage your submitted products:

- Browse all products you've submitted
- Filter by status, category, or batch
- View product details and history
- Track which products have been synced

Learn more: [Product Management](./vendor/product-management.md)

## Getting Help

### Support Resources

- **Email Support**: support@vendorportal.com
- **Response Time**: Within 24 hours on business days
- **Business Hours**: Monday - Friday, 9:00 AM - 5:00 PM EST

### Before Contacting Support

1. Check the [Troubleshooting Guide](./common/troubleshooting.md) for common issues
2. Review the relevant documentation section
3. Gather information about your issue:
   - What were you trying to do?
   - What happened instead?
   - Any error messages displayed?
   - Screenshots (if applicable)

### Reporting Issues

When reporting a bug or issue, please include:

- Browser and version
- Steps to reproduce the issue
- Expected vs. actual behavior
- Screenshots or error messages
